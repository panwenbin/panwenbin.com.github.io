对齐#

Python的格式特点是缩进对齐，优点是整齐得到了语言层面的保证，缺点是缩进多了其实容错率低，特别是Tab和空格在视觉上不容易区分。

类型#

  • Numbers（数字）
  • String（字符串）
  • List（列表）
  • Tuple（元组）
  • Dictionary（字典）

变量#

Python的变量不需要声明，直接赋值即可。变量的类型由赋值决定。

字符串#

单引号和双引号是一样的，三引号可以用来表示多行字符串。

list#

使用[和]来表示列表，列表可以包含不同类型的元素。

使用:来切片。

tuple#

使用(和)来表示元组，元组是不可变的。

dict#

使用{}来表示字典，字典是键值对的集合。
